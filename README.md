# Podkop Tools

Менеджер обслуживания Podkop для OpenWrt.

Проект объединяет в одном интерактивном меню наиболее востребованные операции по обслуживанию Podkop, автоматически используя официальные установочные скрипты проектов и добавляя собственные инструменты автоматизации.

## Возможности

- Обновление sing-box до актуальной версии
- Применение xHTTP Patch
- Установка автоматического обновления Podkop
- Установка собственного curl-патча для загрузки удалённых списков
- Перезапуск Podkop
- Проверка состояния (`podkop global_check`)
- Запуск полного обслуживания одной командой
- Автоматическая очистка временных файлов
- Проверка корректности скачанных скриптов перед выполнением
- Поддержка локальных пользовательских патчей (`patches/*.sh`)

## Используемые проекты

Podkop Tools не содержит собственных версий этих проектов, а использует их официальные установочные скрипты.

- Podkop — https://github.com/itdoginfo/podkop
- OpenWRT sing-box Extended — https://github.com/EikeiDev/OpenWRT-sing-box-extended
- Podkop xHTTP Patch — https://github.com/moix89/podkop-xhttp-patch
- Podkop Auto Update — https://github.com/irat25/podkop-auto-update

## Установка

```sh
wget -qO /tmp/install.sh https://raw.githubusercontent.com/Vixald/podkop-tools/main/install.sh && sh /tmp/install.sh
```

## Меню

```
1) Update sing-box
2) Apply patches
3) Update sing-box + Apply patches
4) Restart Podkop
5) Podkop global_check
0) Exit
```

## Архитектура

Podkop Tools выполняет роль единой оболочки (wrapper) над несколькими независимыми проектами.

Во время работы необходимые официальные установочные скрипты автоматически скачиваются с GitHub, проходят проверку и запускаются. После выполнения временные файлы удаляются.

Локальные пользовательские патчи можно разместить в каталоге `patches/`. Если каталог отсутствует, этот этап автоматически пропускается.

## Требования

- OpenWrt
- Podkop
- curl или wget
- доступ к GitHub

## Лицензия

Каждый внешний проект распространяется по собственной лицензии.

Podkop Tools содержит только код, относящийся к объединению и автоматизации этих проектов.
